FORMULARIO REPORTE DE COMISIONES

Herramienta para consultar y totalizar las comisiones de un vendedor en un rango de fechas especifico. Extrae datos mediante la union de las tablas de Vendedores y Comisiones.

GUIA DE USO

 

ID Vendedor: Ingrese el codigo numerico unico del empleado.

Fecha Inicial/Final: Defina el periodo de tiempo.

Formato obligatorio: AAAA-MM-DD (ej. 2026-05-01).

Boton Consultar: Ejecuta la busqueda en la base de datos y actualiza la tabla de resultados.

 

ESTRUCTURA DE RESULTADOS

 

La tabla muestra la informacion procesada en cuatro columnas:

#Vendedor: Identificador unico.

Vendedor: Nombre completo (obtenido via JOIN).

Fecha De Registro: Dia en que se genero la comision.

Comision: Monto economico obtenido.

Fila de Total: Al final de la lista, el sistema suma automaticamente todas las comisiones del periodo.

FORMULARIO PROCESO COMISIONES VENTAS

¿Que hace este formulario? Este formulario permite consultar y calcular las comisiones de ventas de un empleado especifico. A traves de su ID, carga la informacion del empleado y muestra un desglose detallado de sus comisiones segun diferentes criterios de visualizacion (por vendedor, linea, marca o producto), incluyendo datos como metas, marcas, metas adicionales y comision generada, para finalmente ejecutar el calculo total de las comisiones correspondientes.

Guia de Uso

ID Empleado: Ingrese el codigo numerico unico del empleado del cual desea consultar las comisiones.

Boton Cargar datos: Una vez ingresado el ID, presione este boton para cargar la informacion del empleado y poblar el campo de nombre automaticamente.

Nombre Empleado: Campo de solo lectura que se completa automaticamente al cargar los datos, mostrando el nombre completo del empleado asociado al ID ingresado.

Opciones de vista (radio buttons): Seleccione el nivel de detalle con el que desea visualizar las comisiones:

  • Vendedores: Muestra las comisiones agrupadas por vendedor.
  • Linea: Desglosa las comisiones por linea de productos.
  • Marca: Presenta las comisiones organizadas por marca.
  • Producto: Muestra el detalle de comisiones a nivel de producto individual.

Tabla de resultados: Despliega la informacion cargada con las columnas ID, Empleado, Comisiones ventas, Meta, Marca, Metas adicionales y Comision, segun el filtro de vista seleccionado.

Boton Calcular: Ejecuta el calculo de las comisiones con base en los datos cargados y el criterio de vista seleccionado, mostrando el resultado total en el campo de texto ubicado a su derecha.
