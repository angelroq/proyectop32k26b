Instructivo del modulo compras

Registro de compras

Paso 1: Llenar los campos del formulario

Ingrese los datos solicitados en el formulario de compras. Los campos grises no se deben escribir manualmente, porque el sistema los calcula o los genera de forma automatica.

La siguiente tabla muestra los campos del formulario con nombres sencillos y el tipo de dato que se debe ingresar.

Campo del formulario Tipo de dato aceptado Explicacion
ID de factura Numero entero Identifica una factura ya registrada. Normalmente lo genera el sistema.
Numero de factura Letras, numeros y signos basicos Codigo o numero escrito en la factura. No debe repetirse.
ID de proveedor Numero entero Codigo del proveedor registrado en el sistema.
Estado Letras, numeros y signos basicos Estado de la factura. Ejemplo: Vigente, Anulada o Pendiente.
Subtotal de factura Numero decimal Campo gris. Se calcula automaticamente con base en cantidad y precio.
IVA Numero decimal Campo gris. Se calcula automaticamente.
Total Numero decimal Campo gris. Muestra la suma del subtotal mas el IVA.
ID de detalle Numero entero Identifica un detalle ya registrado. Normalmente lo genera el sistema.
ID de producto Numero entero Codigo del producto registrado en el sistema.
Cantidad de producto Numero decimal Cantidad comprada del producto.
Precio unitario de producto Numero decimal Precio de una unidad del producto.
Tabla de facturas Solo lectura Muestra las facturas registradas.
Tabla de detalles Solo lectura Muestra los detalles de las facturas registradas.
Nota: El ID de proveedor y el ID de producto deben existir previamente en el sistema. Si no existen, la compra no podra registrarse.

Paso 2: Presionar el boton Calcular gastos

Despues de ingresar el precio unitario y la cantidad del producto, presione el boton Calcular gastos. El sistema calculara el subtotal, el IVA y el total.

Paso 3: Presionar el boton Registrar

Despues de revisar que los datos sean correctos, presione Registrar. Esta accion guarda la factura y su detalle en la base de datos.

Consulta, cambios y eliminacion

Paso 1: Escribir el ID correspondiente

Para buscar, actualizar o eliminar una factura, escriba el ID de factura en el campo disponible para ello.

Si desea actualizar un detalle especifico, escriba tambien el ID de detalle. Este ID se puede consultar en la tabla de detalles.

Paso 2: Presionar el boton requerido

Luego de escribir el ID necesario, presione el boton de la operacion deseada.

Boton Funcion
Buscar Busca una factura por su ID y muestra sus datos en el formulario.
Actualizar Modifica los datos de la factura y del detalle indicado.
Eliminar Elimina la factura seleccionada y sus detalles relacionados.
Reporte Genera o muestra el reporte del modulo, si esta disponible.
Help Abre este instructivo de ayuda.
Importante: Antes de actualizar o eliminar, verifique que el ID ingresado corresponda al registro correcto.
